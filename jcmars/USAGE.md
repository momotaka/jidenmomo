## 基本的な使い方

### 1. システムへのアクセス

1. Webブラウザを開きます（推奨: Chrome, Firefox, Safari, Edge）
2. アドレスバーに以下のURLを入力します：
   - ローカル環境: `http://localhost`
   - 本番環境: `https://your-domain.com`

### 2. トップページの構成

```
┌─────────────────────────────────────────┐
│  JCMARS                                 │
│  JC役職者自動ランキングシステム            │
│  「役職は覚えているが名前が思い出せない」    │
│  を解決します                            │
├─────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────────┐  │
│  │ 年度        │  │ 地域名          │  │
│  │ [2024    ] │  │ [新潟ブロック  ] │  │
│  └─────────────┘  └─────────────────┘  │
│  ┌─────────────┐  ┌─────────────────┐  │
│  │ 役職        │  │ 追加キーワード   │  │
│  │ [会長     ] │  │ [          ]   │  │
│  └─────────────┘  └─────────────────┘  │
│         [ 思い出す！ ]                  │
└─────────────────────────────────────────┘
```

### 3. 検索手順

#### Step 1: 年度の入力
- 検索したい年度を入力します
- 範囲: 1990年〜2030年
- デフォルト: 現在の年度

#### Step 2: 地域名の入力
- 地域、ブロック、青年会議所名を入力します
- 入力補助機能で候補が表示されます
- 例: 
  - 地区レベル: "関東地区協議会", "東北地区協議会"
  - ブロックレベル: "新潟ブロック", "千葉ブロック"
  - LOMレベル: "新潟青年会議所", "東京青年会議所"

#### Step 3: 役職の入力
- 検索したい役職を入力します
- 入力補助機能で候補が表示されます
- 主な役職:
  - 会長系: "会頭", "会長", "理事長"
  - 副会長系: "副会頭", "副会長", "副理事長"
  - 理事系: "専務理事", "常務理事", "理事"
  - 委員会系: "委員長", "副委員長", "委員"
  - 事務局系: "事務局長", "財政局長"

#### Step 4: 追加キーワード（任意）
- より詳細に絞り込みたい場合に使用
- 例: "総務委員会", "青少年育成", "地域開発"

#### Step 5: 検索実行
- 「思い出す！」ボタンをクリック
- 検索には10〜60秒程度かかります

## 検索機能の詳細

### 地域名の入力パターン

#### 1. 完全一致検索
正式名称で入力すると最も正確な結果が得られます：
- ✅ "新潟ブロック協議会"
- ✅ "関東地区協議会"
- ✅ "新潟青年会議所"

#### 2. 部分一致検索
略称や通称でも検索可能です：
- ✅ "新潟ブロック" → "新潟ブロック協議会"として検索
- ✅ "新潟JC" → "新潟青年会議所"として検索
- ✅ "関東地区" → "関東地区協議会"として検索

#### 3. 都道府県名での検索
都道府県名を含めることで、より正確な検索が可能：
- ✅ "東京都" + "理事長" → 東京都内のLOMの理事長
- ✅ "新潟県" + "会長" → 新潟県内の組織の会長

### 役職の階層構造

```
日本青年会議所（全国）
├── 会頭
├── 副会頭
└── 専務理事

地区協議会（10地区）
├── 会長
├── 副会長
└── 専務理事

ブロック協議会（都道府県）
├── 会長
├── 副会長
└── 理事長（一部）

LOM（市区町村）
├── 理事長
├── 副理事長
└── 専務理事
```

### 検索の仕組み

1. **年度範囲の自動拡張**
   - 指定年度の前後1年も検索対象に含めます
   - 例: 2023年を指定 → 2022年、2023年、2024年を検索

2. **複数情報源の確認**
   - 最大20サイトから情報を収集
   - 公式サイトを優先的に評価

3. **信頼度スコアリング**
   - 情報源の信頼性（50%）
   - 情報の完全性（30%）
   - 複数ソースでの確認（20%）

### 各項目の説明

#### 1. ランキング番号
- 信頼度の高い順に番号が付与されます
- 同じ信頼度の場合は、情報源の数で順位決定

#### 2. 候補者情報
- **名前**: 検出された人物名
- **年度**: 役職に就いていた年度
- **組織名**: 所属組織の正式名称
- **役職**: 具体的な役職名

#### 3. 信頼度スコア
- **90%以上**: 非常に信頼できる（公式サイト等で確認）
- **70-89%**: 信頼できる（複数の信頼できるソースで確認）
- **60-69%**: ある程度信頼できる（少なくとも1つの信頼できるソースで確認）
- **60%未満**: 表示されない（最低信頼度でフィルタリング）

#### 4. 情報源
- 情報が確認されたWebサイトのリスト
- リンクをクリックすると該当ページが開きます
- より多くの情報源がある場合は信頼度が高い

#### 5. 抽出コンテキスト
- 実際に情報が抽出された文章の一部
- 前後の文脈を含めて表示

### 特殊な表示

- 日本青年会議所の公式サイトで確認された情報

## 効果的な検索のコツ

### 1. 正確な地域名を使用する

❌ 悪い例:
- "新潟" → 範囲が広すぎる
- "ブロック" → 特定できない

✅ 良い例:
- "新潟ブロック"
- "新潟ブロック協議会"
- "上越青年会議所"

### 2. 役職名は正式名称で

❌ 悪い例:
- "トップ" → 曖昧
- "リーダー" → JC用語ではない

✅ 良い例:
- "理事長"
- "会長"
- "委員長"

### 3. 年度は正確に

- 任期は通常1年間（1月〜12月）
- 年度を間違えると検索結果が出ない可能性

### 4. 追加キーワードの活用

特定の委員会や事業に関わっていた場合：
- "総務委員会 委員長"
- "青少年育成委員会"
- "会員拡大"

### 5. 検索範囲の理解

```
広い ← → 狭い
地区協議会 > ブロック協議会 > LOM（青年会議所）
```

- より狭い範囲で検索すると精度が上がります
- 広い範囲で検索すると候補が多くなります

## よくある質問（FAQ）

### Q1: 検索に時間がかかるのはなぜですか？

A: JCMARSはリアルタイムでWeb上の情報を検索・分析しています。以下の処理を行うため時間がかかります：
1. Google検索で関連ページを探索（5-10秒）
2. 各ページの内容を取得・分析（20-40秒）
3. 信頼度計算とランキング（5-10秒）

### Q2: 検索結果が出ない場合は？

A: 以下を確認してください：
1. **年度が正しいか**: ±1年ずらしてみる
2. **地域名が正しいか**: 正式名称を使用する
3. **役職名が正しいか**: JCで使われる正式な役職名を使用
4. **情報が公開されているか**: 古い年度や小規模LOMは情報が少ない場合がある

### Q3: 信頼度が低い結果しか出ない場合は？

A: 以下の理由が考えられます：
1. 公式サイトに情報が掲載されていない
2. 検索対象の年度が古い
3. 地域名や役職名が正確でない

対策：
- より具体的なキーワードを追加
- 年度を前後にずらして検索
- 地域名を正式名称に変更

### Q4: 同じ人が複数表示される場合は？

A: 以下のケースがあります：
1. 複数年度で同じ役職に就いていた
2. 異なる組織で役職を務めていた
3. 表記揺れ（漢字の異体字など）

### Q5: キャッシュとは何ですか？

A: 一度検索した結果を一時的に保存する機能です：
- 同じ検索を60分以内に行うと高速で結果を表示
- 新しい情報は反映されない可能性がある
- 最新情報が必要な場合は60分以上待つか、検索条件を少し変更

### Q6: スマートフォンでも使えますか？

A: はい、使用できます：
- レスポンシブデザイン対応
- タッチ操作に最適化
- 画面サイズに応じて表示を調整

### Q7: 検索履歴は保存されますか？

A: いいえ、プライバシー保護のため：
- 個人の検索履歴は保存されません
- ログインは不要
- 個人情報の収集はありません

### Q8: エラーが表示された場合は？

A: 以下を試してください：
1. ページを再読み込み（F5キー）
2. ブラウザのキャッシュをクリア
3. 別のブラウザで試す
4. しばらく時間をおいて再度アクセス

## 用語集

### JC関連用語

**JC（ジェイシー）**
- Junior Chamber（青年会議所）の略称
- 20歳から40歳までの青年の団体

**LOM（ロム）**
- Local Organization Memberの略
- 各地域の青年会議所の単位

**日本JC**
- 公益社団法人日本青年会議所
- 全国のJCを統括する組織

**地区協議会**
- 全国を10地区に分けた地域組織
- 複数のブロック協議会を統括

**ブロック協議会**
- 都道府県レベルの組織
- 複数のLOMを統括

### 役職用語

**会頭（かいとう）**
- 日本青年会議所のトップ
- 1年任期

**理事長（りじちょう）**
- 各LOMのトップ
- 組織の代表者

**会長（かいちょう）**
- 地区協議会、ブロック協議会のトップ

**専務理事（せんむりじ）**
- 組織運営の実務責任者

**委員長（いいんちょう）**
- 各委員会の責任者
- 特定の事業や活動を担当

### システム用語

**信頼度スコア**
- 情報の確からしさを示す指標
- 0〜100%で表示

**キャッシュ**
- 検索結果の一時保存
- 高速化のための仕組み

**コンテキスト**
- 抽出された情報の前後の文脈
- 情報の正確性を確認するため

## 検索のベストプラクティス

### 初心者向け

1. まず年度を正確に入力
2. 地域名は入力補助から選択
3. 役職も入力補助から選択
4. 追加キーワードは使わない

### 中級者向け

1. 地域名は正式名称を使用
2. 複数の役職で検索してみる
3. 年度を前後にずらして検索
4. 追加キーワードで絞り込み

### 上級者向け

1. 組織階層を理解して検索
2. 役職の正式名称を把握
3. 複数の検索を組み合わせ
4. 信頼度スコアの意味を理解
